# AgroControl

Sistema de gestión agrícola para la empresa Campos Fértiles S.A., desarrollado en Python usando Tkinter para la interfaz gráfica.

## ¿Qué hace?

La aplicación tiene 7 módulos organizados en pestañas:

- Fincas
- Cultivos
- Insumos
- Maquinaria
- Empleados
- Cosechas
- Clientes

Cada módulo tiene un formulario con los campos correspondientes y botones para guardar, actualizar, eliminar y limpiar. La interfaz está preparada para conectarse a una base de datos, pero por ahora no incluye CRUD funcional.

## Requerimientos 

- Python 3.8 o superior
- Tkinter (viene incluido con Python)

En Linux, si no tienes Tkinter instalado:
```bash
sudo apt-get install python3-tk
```

## ejecutarlo

```bash
git clone https://github.com/tu-usuario/agrocontrol.git
cd agrocontrol
python agrocontrol.py
```

## Tecnologías usadas
